# Lecoo Control Center
**Created for Lecoo Pro 14 (Lecoo N155). WIP.**

Lecoo Control Center is a reverse-engineered, low-level Embedded Controller (EC) daemon and client suite designed for laptops based on the Emdoor chassis (e.g., Lecoo Pro 14, maybe even others, have no clue lol). It provides direct hardware-level control over cooling, power limits, and lighting. Because there is no official software for this!
